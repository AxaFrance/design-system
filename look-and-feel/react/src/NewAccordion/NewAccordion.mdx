# NewAccordion

## Import

```tsx
import { NewAccordion } from "@axa-fr/design-system-look-and-feel-react";
```

## Uses

You can force to open or close the accordion with the `isOpen` prop. (if no `onClick` prop is given the component will work like natively)

And when the user clicks on the summary to open or close the accordion you can use the `onClick` function prop to do something at that moment. (you will get the native event as `onClick` parameter)

You can change the `className` or add a `classModifier` if you want to custom the component.

You can also add/change the props of the `Tag` component inside of the `NewAccordion`.
