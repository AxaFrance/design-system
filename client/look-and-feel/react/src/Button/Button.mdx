# Button

To use the button import it like that:

```tsx title="test.js"
import {
  ButtonClient,
  Variants,
} from "@axa-fr/design-system-look-and-feel-react";

const MyComponent = () => (
  <ButtonClient variant={Variants.secondary}>My button</ButtonClient>
);
```

## Variants

Use the `variant` prop to change the look of the button. [See the examples](#examples-of-class-modifiers).

You can also add your own modifier, and the output will be a custom class called `af-btn-client--<yourModifier>` that you can target in your css
The rest of the props (eg: `type`, `title`) will be passed down to the actual `button` HTML element.
