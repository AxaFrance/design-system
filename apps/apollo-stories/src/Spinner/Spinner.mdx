# Spinner

To use the spinner import it like that:

```tsx title="test.js"
import { Spinner } from "@axa-fr/design-system-apollo-react";

const MyComponent = () => (
  <Spinner size={60} variant="gray" text="Chargement en cours" />
);
```

## Variants

Use the `variant` prop to change the look of the spinner. The available variants are `"blue"` (default), `"gray"` and `"white"`. [See the examples](#examples-of-variants).
