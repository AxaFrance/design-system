# CardCheckboxOption components

## How to use

To use the `CardCheckboxOption`component import it like this:

```tsx

const MyComponent = () => (
  <CardCheckboxOption
    label="Titre"
    description="Sous-titre 1"
    subtitle="Sous-titre 2"
    icon={icon}
    name="foo"
    value="bar"
  />

```
