# Dropdown

To use Dropdown import it like this:

```tsx
import { Dropdown } from "@axa-fr/canopee-react/prospect";

const MyComponent = () => <Dropdown label="label">
```

## Type

Use the `type` prop to change the type of the select. The available variants are `default`, `error` and `disabled`.
