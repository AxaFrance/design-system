# Select

To use Select import it like this:

```tsx
import { Select } from "@axa-fr/design-system-apollo-react";

const MyComponent = () => <Select label="label">
```

## Type

Use the `type` prop to change the type of the select. The available variants are `default`, `error` and `disabled`.
