# TableCard

To use the table card import it like that:

```tsx
import { TableCard } from "@axa-fr/canopee-react/prospect";

const MyComponent = () => (
  <TableCard variant="alternate">
    <TableCard.DRow>
      <TableCard.Dt>Produit/Support</TableCard.Dt>
      <TableCard.Dd>AB Sustainable Global Thematic A</TableCard.Dd>
    </TableCard.DRow>
    <TableCard.DRow>
      <TableCard.Dt>Code Isin</TableCard.Dt>
      <TableCard.Dd>LU0101010101</TableCard.Dd>
    </TableCard.DRow>
    <TableCard.DRow>
      <TableCard.Dt>Status</TableCard.Dt>
      <TableCard.Dd>Ouvert</TableCard.Dd>
    </TableCard.DRow>
  </TableCard>
);
```

## Usage

The Table component is a compound component with the following sub-components:

- `TableCard.DRow` - TableCard row wrapper with optional `direction` prop
- `TableCard.Dt` - Table row title
- `TableCard.Dd` - Table row description
