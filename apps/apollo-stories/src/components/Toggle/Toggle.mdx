# Toggle

To use the toggle import it like that:

```tsx
import { Toggle } from "@axa-fr/design-system-apollo-react";

const MyComponent = () => (
  <>
    <Toggle checked={checked} disabled={disabled} onChange={onChange} />
    <Toggle ref={checkboxRef} />
    <Toggle checked disabled />
  </>
);
```
