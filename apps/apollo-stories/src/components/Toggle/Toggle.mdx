# Toggle

To use the toggle import it like that:

```tsx
import { Toggle } from "@axa-fr/design-system-apollo-react";

const MyComponent = () => (
  <>
    <Toggle selected={false} handleChange={setSelected} />
    <Toggle selected={true} disabled={true} />
  </>
);
```
