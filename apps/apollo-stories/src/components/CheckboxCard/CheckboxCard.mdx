# CheckboxCard components

## CheckboxCard

### How to use

To use the `CheckboxCard`component import it like this:

```tsx
import { CheckboxCard } from "@axa-fr/design-system-apollo-react";

const MyComponent = () => (
  <CheckboxCard
    name="name"
    value="value"
    label="J'accepte de fournir à AXA mes coordonnées ainsi que les données  relatives à mon projet et ma situation. Ces dernières seront transmises à mon conseiller AXA qui pourra  me contacter pour m'accompagner."
  />
);
```
