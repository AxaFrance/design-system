# Tag

To use the tag import it like that:

```tsx
import { Tag } from "@axa-fr/design-system-apollo-react";

const MyComponent = () => (
  <>
    <Tag variant="info">My tag</Tag>
    <Tag variant="success">My tag</Tag>
  </>
);
```
