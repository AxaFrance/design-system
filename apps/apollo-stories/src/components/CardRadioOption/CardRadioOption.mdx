# CardRadioOption components

## How to use

To use the `CardRadioOption`component import it like this:

```tsx
import { CardRadioOption } from "@axa-fr/design-system-apollo-react";

const MyComponent = () => (
  <CardRadioOption
    label="Titre"
    description="Sous-titre 1"
    subtitle="Sous-titre 2"
    icon={icon}
    name="foo"
    value="bar"
  />
);
```
