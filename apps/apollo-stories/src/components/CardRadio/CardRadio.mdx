# CardRadio components

## CardRadio

### How to use

To use the `CardRadio`component import it like this:

```tsx
import { CardRadio } from "@axa-fr/design-system-apollo-react";

const MyComponent = () => (
  <CardRadio
    type="vertical"
    options={[
      {
        label: "Paris",
        description: "Capitale de la France",
        subtitle: "Nord",
        name: "cities",
        value: "paris",
        icon: <Home />,
      },
      {
        label: "Bruxelles",
        description: "Capitale de la Belgique",
        name: "cities",
        value: "bruxelles",
        icon: <Home />,
      },
      {
        label: "Lille",
        name: "cities",
        value: "lille",
        icon: <Home />,
        disabled: true,
        checked: true,
      },
    ]}
  />
);
```

### Modes
