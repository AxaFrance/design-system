# RadioCard components

## RadioCard

### How to use

To use the `RadioCard`component import it like this:

```tsx
import { RadioCard } from "@axa-fr/design-system-apollo-react";

const MyComponent = () => (
  <RadioCard
    type="vertical"
    options={[
      {
        label: "Paris",
        description: "Capitale de la France",
        subtitle: "Nord",
        name: "cities",
        value: "paris",
        icon: <Home />,
      },
      {
        label: "Bruxelles",
        description: "Capitale de la Belgique",
        name: "cities",
        value: "bruxelles",
        icon: <Home />,
      },
      {
        label: "Lille",
        name: "cities",
        value: "lille",
        icon: <Home />,
        disabled: true,
        checked: true,
      },
    ]}
  />
);
```

### Modes

The RadioCard component has 2 modes: `vertical` (default) and `horizontal`.
