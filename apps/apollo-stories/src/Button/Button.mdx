# Button

To use the button import it like that:

```tsx
import { Button } from "@axa-fr/design-system-apollo-react";

const MyComponent = () => (
  <>
    <Button>My button</Button>
    <Button variant="secondary">My button</Button>
  </>
);
```

## Variants

Use the `variant` prop to change the look of the button. [See the examples](#all-variants).

You can also add your own modifier, and the output will be a custom class called `af-btn-client--<yourModifier>` that you can target in your css
The rest of the props (eg: `type`, `title`) will be passed down to the actual `button` HTML element.
