## Typography

### Prerequisite

To use the typography, you must install the font "Source Sans Pro" in your project.

#### Install

```bash
npm install @fontsource/source-sans-pro
```

#### Use

```js
import "@fontsource/source-sans-pro/400.css";
import "@fontsource/source-sans-pro/600.css";
import "@fontsource/source-sans-pro/700.css";
```

### Import

#### Typography only

```js
import "@axa-fr/design-system-slash-react/dist/Typography/Typography.scss"; // or
import "@axa-fr/design-system-slash-react/dist/Typography/Typography.css";
```

#### Global

```js
import "@axa-fr/design-system-slash-react/dist/slash.scss"; // or
import "@axa-fr/design-system-slash-react/dist/slash.css";
```

### Use

```html
<h1 class="af-typography--h1">Title</h1>
<p class="af-typography--text">Content</p>
```
