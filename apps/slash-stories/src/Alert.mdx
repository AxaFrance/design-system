## Alert

### Import

```tsx title="test.js"
import { Alert } from "@axa-fr/design-system-slash-react/";
```

### Use

```tsx title="test.js"
export const AlertComponent = () => (
  <Alert classModifier="danger" title="Warning: information is missing" />
);
```

The property `closeButtonAriaLabel` allows projects to change the aria-label of the close button, making it more accessible.

### Modifiers

There are 4 modifiers available for the Alert component, `error` modifier is the default one :
