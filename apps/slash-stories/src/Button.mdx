# Button

To use the button import it like that:

```tsx title="test.js"
import { Button } from "@axa-fr/design-system-slash-react";

const MyComponent = () => (
  <Button classModifier="reverse" type="submit">
    My button
  </Button>
);
```

## ClassModifier

Use the `classModifier` prop to change the look of the button.
[See the examples](#examples-of-class-modifiers).

Its value is a list of space separated values which can be:

- `hasiconLeft`/`hasiconRight` to add an icon on the left or right of the text
- `disabled`
- `success`
- `danger`
- `reverse`
- `small`
- `circle`
- `circle-menu`
- `circle-reverse`
- `circle-small`

You can also add your own modifier, and the output will be a custom class called
`af-btn--<yourModifier>` that you can target in your css The rest of the props
(eg: `type`, `title`) will be passed down to the actual `button` HTML element.

## With icons

To add an icon you need to add a `classModifier`, either `hasiconLeft` or
`hasiconRight` whether you want the icon on the left of or on the right. The
text of the button needs to be wrapped in a `span` with the class
`af-btn__text`.
