## Button

### Import

```tsx
import { Button } from "@axa-fr/design-system-slash-react";
```

### Use

#### Button Example

```tsx
const SubmitButton = () => (
  <Button type="submit" variant="secondary">
    My Submit button
  </Button>
);
```

## Button variation

The button has 6 variants, `primary` (default variant), `secondary`, `validated`, `danger`, `ghost` and `ghost-reverse`.\
To choose them, simply use the variant `props`.

### Primary

The Primary CTA component allows you to create one and only one primary action within: a page, a horizontal step train page, a vertical step train section or a modal.

## Button With icons

It is possible to add an icon on the right or left via the `leftIcon` and `rightIcon` props.\
This props expects a react component, you can use an svg or a glyphicon component.
