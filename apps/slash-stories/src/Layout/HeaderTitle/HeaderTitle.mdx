# HeaderTitle

By default, the title bar is sticky on the top when you scroll down the page. If you want to disable that behavior you can set to false the `isSticky` property of the component.

## Import

```tsx
import { HeaderTitle } from "@axa-fr/canopee-react/distributeur";
```

## Use

```tsx
const TitleComponent = () => (
  <HeaderTitle
    isSticky
    title="Toolkit Axa"
    subtitle="Info complémentaire"
    toggleMenu={() => {}}
  />
);
export default TitleComponent;
```

If you need to have a navbar with anchors, you can use the `anchorNavBarItems` property. The `HeaderTitle` component will then show the `AnchorNavBar` component with the items you provide. You should update the `isActive` property of the items when the user clicks on them to highlight the active item. The `onClick` function will be called when the item is clicked, allowing you to handle navigation or other actions.
The `anchorNavBarItems` property is an array of objects with the following structure:

```tsx
interface AnchorNavBarItem {
  name: string; // The name of the item
  isActive?: boolean; // Optional, if true, the item will be highlighted as active
  onClick?: () => void; // Optional, if provided, will be called when the item is clicked
  link?: string; // The link to navigate to
  externalLink?: boolean; // Optional, if true, an icon will be displayed to indicate that the link is external
  render?: (props: any) => React.ReactNode; // Optional, if provided, will render the item using the provided function
}
```

## Use

```tsx
const TitleComponent = () => (
  <HeaderTitle
    isSticky
    title="Toolkit Axa"
    subtitle="Info complémentaire"
    toggleMenu={() => {}}
    anchorNavBarItems={[
      { name: "Home", link: "#home" },
      { name: "About", link: "#about" },
      { name: "Services", link: "/services", externalLink: true },
      {
        name: "Contact",
        render: (props) => (
          <a {...props} href="#contact">
            Contact
          </a>
        ),
      },
    ]}
  />
);
export default TitleComponent;
```

### Complex Title with Content and Children

When you need to handle a navigation action , a back button for example, you can use the `contentLeft` property. For other others actions you should put them the right side of the title bar with the `contentRight` property. Passing children allows you to add additional content after the title and subtitle.
The properties accepts a ReactNode to give you the flexibility to add any content you want.
