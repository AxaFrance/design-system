## Message

### Import

```tsx title="App.tsx"
import { Message } from "@axa-fr/canopee-react/distributeur/";
```

### Use

```tsx title="App.tsx"
export const MessageComponent = () => (
  <Message classModifier="danger" title="Warning: information is missing" />
);
```

The property `closeButtonAriaLabel` allows projects to change the aria-label of the close button, making it more accessible.

### Modifiers
