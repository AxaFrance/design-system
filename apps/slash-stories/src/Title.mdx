## Title

### Use of Title

```tsx title="test.js"
import { Title } from "@axa-fr/design-system-slash-react";

export const MyComponent = () => (
  <Title classModifier="af-title" className="">
    Sample Title
  </Title>
);
```

The title can also contain any React component. Simply pass it into :

- the `contentLeft` prop if you want it to be displayed between the title and the separator.
- the `contentRight` prop if you want it to be displayed after the separator.

```tsx title="content.js"
import { Title } from "@axa-fr/design-system-slash-react";

export const MyComponent = () => (
  <Title
    classModifier="af-title"
    className=""
    contentLeft={<Button>Left button</Button>}
    contentRight={<Button>Right button</Button>}
  >
    Title with content
  </Title>
);
```
