# Spinner

The spinner is a component to show a loading indicator in your project.
Three variants are available:

- `inline`: a one-line spinner. This is the default variant.
- `fullScreen`: a fullscreen spinner.
- `content`: same shape as the fullScreen spinner but with a smaller icon, intended for content areas.

```tsx
import { Spinner } from "@axa-fr/canopee-react/distributeur";

const MyComponent = () => (
  <Spinner variant="inline" text="Recherche en cours" />
);
```
