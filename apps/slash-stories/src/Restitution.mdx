## Restitution

The Restitution components are a group of related components that are used to
display recapitulative data. The main component is `ArticleRestitution` which is
a container for the other components.

Then inside the `ArticleRestitution` you can have a `HeaderRestitution` which is
the header of the restitution. The `HeaderRestitution` can have a `title`, a
`subtitle` and a `rightTitle` which is a link to edit the restitution.

Then you can have a `SectionRestitution` which is a container for the
`SectionRestitutionRow` or `SectionRestitutionColumn`. These last two can
include each other depending on the layout you want. Finally you can have a
`Restitution` which is a label and a value, and a `RestitutionList` which is a
list of values.

### Import

```tsx
import {
  ArticleRestitution,
  HeaderRestitution,
  SectionRestitution,
  SectionRestitutionRow,
  SectionRestitutionColumn,
  Restitution,
  RestitutionList,
} from "@axa-fr/design-system-react/agent";
```
