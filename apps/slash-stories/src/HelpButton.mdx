# HelpButton

The `HelpButton` component is the easiest way to create tooltips in the slash design-system.
It wraps around the more customisable `Popover` component with defaults styles. By default the element to be hovered/clicked is a blue circle button with a cursive `i` inside.

## Usage

```tsx title="test.js"
import { HelpButton } from "@axa-fr/design-system-slash-react";

const MyComponent = () => (
  <HelpButton mode="hover">Content of the tooltip</HelpButton>
);
```

## Customize

If you need rich content inside the tooltip, it is possible to pass HTML as children of the component.
You can also replace the blue circle with custom HTML by changing the `helpButtonContent` prop.
