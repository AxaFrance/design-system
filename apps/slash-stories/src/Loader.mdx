# Loader

> **Breaking-change safe migration**: `Loader` now supports variants. The default variant keeps the historical Loader behavior.
>
> **Deprecated component**: `SquareLoader` is still exported for compatibility but is deprecated. Prefer `Loader` with `variant="fullscreen"`.
>
> **Deprecated prop**: The `classModifier` prop is deprecated. Use `className` to add custom CSS modifier classes instead.

The loader is a component to disable part of your application to express an unspecified wait time or display the length of a process.

The default variant is the `fullscreen` variant, which is a fullscreen overlay spinner. The `inline` and `content` variants are also available for inline and content area spinners.

For `fullscreen`, you can pass `children` to wrap page content and display the loader overlay above it (same usage pattern as `SquareLoader`).

```tsx
import { Loader } from "@axa-fr/canopee-react/distributeur";

const MyComponent = () => (
  <>
    <Loader variant="fullscreen" text="Chargement en cours">
      My Content
    </Loader>
    <Loader variant="inline" text="Recherche en cours" />
  </>
);
```

## Variants

`Loader` supports 3 variants:

- `inline`: spinner displayed inline
- `content`: compact spinner for content areas
- `fullscreen`: fullscreen overlay spinner

## Deprecated Component

`SquareLoader` is still available and behaves exactly like the previous Loader implementation.
