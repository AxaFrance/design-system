# Loader

The loader is a component to show a loading svg in your project.
Three variants are available:

- `inline`: a one-line loader. This is the default variant.
- `fullScreen`: a fullscreen loader.
- `content`: same shape as the fullScreen loader but with a smaller spinner, intended for content areas.

```tsx
import { Loader } from "@axa-fr/canopee-react/distributeur";

const MyComponent = () => <Loader variant="inline" text="Recherche en cours" />;
```

NB: Previously, the loader wrapped the content shown after loading. This is no longer the case — render the loader and toggle it with your loading state. The previous behavior is still available as `LoaderLegacy`, marked as deprecated, and will be removed in the next major release.
