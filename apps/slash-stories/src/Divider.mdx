# Divider

The `Divider` component is the easiest way to create dividers in the slash design-system.
By default, it renders a horizontal line that can be used to separate content. It can also be styled as a vertical divider.

## Playground

<Canvas of={DividerStories.Default} />
<Controls of={DividerStories.Default} />

## Usage

```tsx
import { Divider } from "@axa-fr/canopee-react/distributeur";

const MyComponent = () => <Divider mode="horizontal" />;
```
