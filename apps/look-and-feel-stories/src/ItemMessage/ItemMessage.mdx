# ItemMessage

To use the spinner import it like that:

```tsx
import { ItemMessage } from "@axa-fr/design-system-look-and-feel-react";
//or
import { ItemMessage } from "@axa-fr/design-system-apollo-react/lf";

const MyComponent = () => <ItemMessage type="success" message="My message" />;
```

## Type

Use the `type` prop to change the type of the item message. The available variants are `error` (default) or `success`.
