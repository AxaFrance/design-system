# Spinner

To use the spinner import it like that:

```tsx
import { Spinner } from "@axa-fr/design-system-look-and-feel-react";
//or
import { Spinner } from "@axa-fr/design-system-apollo-react/lf";

const MyComponent = () => (
  <Spinner size={40} variant="blue" text="Chargement en cours" />
);
```

## Sizes

Use the `size` prop to change the size of the spinner. The available variants are `40` (default), `32` and `24`.

## Variants

Use the `variant` prop to change the look of the spinner. The available variants are `"blue"` (default), `"gray"` and `"white"`. [See the examples](#examples-of-variants).
