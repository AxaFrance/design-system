# Accordion

## Import

```tsx
import { Accordion } from "@axa-fr/design-system-look-and-feel-react";
```

## Uses

You can force to open or close the accordion with the `isOpen` prop. (if no `onClick` prop is given, the component will use the `isOpen` prop as a default value)

When the user clicks on the summary to open or close the accordion you can use the `onClick` function prop to do something at that moment. (you will get the native event as `onClick` parameter)

Two styles of this component are possible, you can use the boolean prop `isTitleFirst` to see them. (default is `true`)

You can change the `className` or add a `classModifier` if you want to custom the component.

You can also add/change the props of the `Tag` component inside of the `Accordion`.
