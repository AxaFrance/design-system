## Skeleton

### Import

```tsx
import { Skeleton } from "@axa-fr/design-system-look-and-feel-react";
```

### Use

```tsx
export const SkeletonComponent = () => <Skeleton grid={[[12]]} />;
```

### grid props

The 'grid' property receives an array of numbers. The first level of the array corresponds to the number of lines.

For a line to be 100%, it must be given a value of 12, which corresponds to the default max value (modifiable by another property);

For example, if you want 3 full lines :

```tsx
export const SkeletonComponent = () => <Skeleton grid={[[12], [12], [12]]} />;
```

If you want 2 lines at 100% and the 3rd at 50% :

```tsx
export const SkeletonComponent = () => <Skeleton grid={[[12], [12], [6]]} />;
```

Multiple cells can be defined within a single line.

If you want 2 rows at 100% and on the 3rd row, 2 cells at 50% :

```tsx
export const SkeletonComponent = () => <Skeleton grid={[[12], [12], [6, 6]]} />;
```

## MaxCol

This property defines the maximum number of columns, with a default value of 12.

```tsx
export const SkeletonComponent = () => (
  <Skeleton maxCols={8} grid={[[8], [8], [4, 4]]} />
);
```

> Note that by changing the max value of columns to 8, to obtain 100% of the value is 8 and no longer 12.

## ColGap

This property defines the spacing between columns (gutters), with a default value of 1. (rem)

```tsx
export const SkeletonComponent = () => (
  <Skeleton colGap={2} grid={[[12], [12], [6, 6]]} />
);
```
