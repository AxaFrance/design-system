# ItemFile

To use the spinner import it like that:

```tsx
import { ItemFile } from "@axa-fr/design-system-apollo-react/lf";

const MyComponent = () => <ItemFile />;
```

## Type

Use the `type` prop to change the type of the item message. The available variants are `error` (default) or `success`.
