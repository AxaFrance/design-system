# CardCheckbox components

## How to use

To use the `CardCheckbox`component import it like this:

```tsx
import { CardCheckbox } from "@axa-fr/canopee-react/prospect";

const MyComponent = () => (
  <CardCheckbox
    type="vertical"
    name="city"
    label="Choisissez des villes"
    options={[
      {
        label: "Paris",
        value: "paris",
        icon: homeIcon,
        description: "Capitale de la France",
        subtitle: "Nord",
      },
      {
        label: "Bruxelles",
        value: "bruxelles",
        icon: homeIcon,
        description: "Capitale de la Belgique",
      },
      {
        label: "Lille",
        value: "lille",
        icon: homeIcon,
        checked: true,
      },
    ]}
  />
);
```

## CardCheckbox with CheckboxText

The CardCheckbox component can also use the CheckboxText for the options.
You have to set the `mode` prop to `text` and provide the options with the `description` and `subtitle` properties.

```tsx
import { CardCheckbox } from "@axa-fr/canopee-react/prospect";

const MyComponent = () => (
  <CardCheckbox
    type="vertical"
    name="city"
    label="Choisissez des villes"
    mode="text"
    options={[
      {
        label: "Paris",
        value: "paris",
        description: "Capitale de la France",
        subtitle: "Nord",
      },
      {
        label: "Bruxelles",
        value: "bruxelles",
        description: "Capitale de la Belgique",
      },
      {
        label: "Lille",
        value: "lille",
        checked: true,
      },
    ]}
  />
);
```
