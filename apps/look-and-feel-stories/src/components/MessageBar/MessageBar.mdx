# Message Bar

To use the message bar import it like that:

```tsx
import { MessageBar } from "@axa-fr/canopee-react/client";

const MyComponent = () => (
  <>
    <MessageBar>My message</MessageBar>
    <MessageBar variant="error">My error message</MessageBar>
  </>
);
```

## Title

The title of the message is set by the `title` prop.
You can set the title level with `heading` prop : "h2" | "h3" | "h4" | "h5" | "h6", the default is "h4".

```tsx
<MessageBar title="My title" heading="h2">
  My message
</MessageBar>
```
