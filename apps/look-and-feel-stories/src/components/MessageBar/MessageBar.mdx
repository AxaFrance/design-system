# MessageBar

A full-width banner used to display important information. Supports a
collapsible description via a toggle chevron.

The colour scheme is driven by the `variant` prop (`blue`, `red`, …). Provide
the icon with the `icon` prop, the optional action through `buttonProps`, and
the initial mobile description state with `defaultDescriptionOpen`. Switch them
in the playground controls below.
