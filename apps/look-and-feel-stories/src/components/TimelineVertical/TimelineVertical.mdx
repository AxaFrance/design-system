# TimelineVertical

To use the timeline vertical import it like that:

```tsx
import { TimelineVertical } from "@axa-fr/canopee-react/client";

const MyComponent = () => (
  <>
    <TimelineVertical
      description="Vous allez recevoir un e-mail d'accès à Sécur'AXA, un espace où transmettre vos documents confidentiels dont vos documents médicaux."
      tag="Étape 1"
      title="Envoi de documents via Sécur'AXA"
    />
  </>
);
```
