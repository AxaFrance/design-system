# React implementation for Slash

Slash is the <abbr title="Business to business">B2B</abbr> design system for AXA France's digital web applications.

## Component documentation

Our UX and UI designers publish documentation and guidelines for each component in:

- [zeroheight](https://zeroheight.com/4b1e27a45/v/latest/p/36b4a2-slash-design-system-b-to-b) for general usage and design guidelines;
- [figma](https://www.figma.com/design/reZserxMfytQ9M82bt20Bi/DS-Slash-V3?node-id=0-1&node-type=canvas&t=X7Uw2gwMfBeD6nJ4-0) for implementation details.

When implementing a new component, you should always refer to these resources first.

## Installation

To install the Slash React package, run:

```bash
npm install @axa-fr/design-system-slash-react
```
