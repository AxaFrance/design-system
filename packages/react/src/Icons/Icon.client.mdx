## Icon

Based on Material UI icons.

### Import

```tsx title="test.js"
import { Icon } from "@axa-fr/design-system-react/icons";
```

### Use

```tsx title="test.js"
export const IconComponent = () => (
  <Icon name="ArrowBack" styleType="filled" size={56} fill="red" />
);
```

### Name

This allow to select the svg to lazy load by the component.
List can be found [here](https://marella.me/material-design-icons/demo/svg/)
