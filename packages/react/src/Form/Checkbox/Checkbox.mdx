## Checkbox

Displays a list of checkbox items. It has several modes you can choose from. See the [modes](#modes) section for more details.
The default mode is `"default"`.

### Import

```javascript
import { Checkbox, CheckboxModes } from "@axa-fr/design-system-react/agent";
```

## CheckboxInput

Displays a list of checkbox items with a global label. It has several modes you can choose from. You can see the different [modes in action in the playground section](#checkboxinput-playground).

### Import

```javascript
import {
  CheckboxInput,
  CheckboxModes,
} from "@axa-fr/react-toolkit-form-input-checkbox";
```

### Modifiers

There are 2 modifiers available for the CheckboxInput component :

<table>
  <tr>
    <th>modifier</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>required</td>
    <td>Makes the input required, and adds the red star next to the label</td>
  </tr>
  <tr>
    <td>disabled</td>
    <td>Disables all the inputs in the component</td>
  </tr>
</table>
