# Accordion

## How to use

To use the `Accordion` component import it like that:

```tsx
import { Accordion } from "@axa-fr/design-system-slash-react/client";

const MyComponent = () => (
  <Accordion>
    <Accordion.Item title="Title 1">Content 1</Accordion.Item>
    <Accordion.Item title="Title 2">Content 2</Accordion.Item>
    <Accordion.Item title="Title 3">Content 3</Accordion.Item>
  </Accordion>
);
```
