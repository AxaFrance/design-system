## Typography

### Import

```tsx
import { Typography } from "@axa-fr/design-system-slash-react";
```

### Use

#### Typography Example:

```tsx
export const TypographyExample = () => (
  <Typography variant="h1">Content H1</Typography>
);
```

#### Playground:

<Canvas of={TypographyStories.TypographyStory} />
<Controls of={TypographyStories.TypographyStory} />

## Customize root element of Typography

The `component` prop overrides the root element, which is by default an `p` tag, and all associated props.

```tsx
import { Typography } from "@axa-fr/design-system-slash-react";

export const Main = () => (
  <Typography component="main" varaint="text">
    Content
  </Typography>
);
```

<Canvas of={TypographyStories.CustomElement} sourceState="shown" />

```tsx
import { Typography } from "@axa-fr/design-system-slash-react";
import { TitleComponent } from "./TitleComponent";

export const Main = () => (
  <Typography component={TitleComponent} variant="h1">
    Title
  </Typography>
);
```
